## 数据结构和算法

### 栈 stack
栈比较简单，直接调用数组的方法模拟。

#### 栈的应用 十进制转换为二进制
思路：10进制数除以二，求余数进栈，然后出栈。

### 队列 queue
跟栈类似，只不过是先进先出

#### 队列的应用
1. 优先队列: 比如医院排队，病人有名字和优先级，然后按照优先级排队插入。
2. 循环队列： 模拟击鼓传花游戏。

构建一个循环队列，给定传花的次数，每次传递 把头元素出队，再入队到尾， 当传递次数结束，头元素出局，如此直到剩下最后一个元素
